LiveWhere API

REST endpoints — Flask + MongoDB · sebl2.pythonanywhere.com

Cities

GET /cities List all cities
GET /cities/:id Get city by ID
POST /cities Create a city
PUT /cities Update population
DEL /cities Delete a city

Cost of Living

GET /cost-of-living COL index for all cities
GET /cost-of-living/salary-adjustment Adjust salary between cities
GET /recommendations Ranked cities by affordability + QOL

Authentication

GET /auth/google Start Google OAuth flow
GET /auth/google/callback OAuth redirect · sets session cookie
GET /auth/me Return current user
POST /auth/logout Clear session · sign out

Utility

GET /hello Health check
GET /endpoints List all routes
GET /swagger.json OpenAPI spec
Flask-RESTX MongoDB Google OAuth 2.0 CORS enabled
